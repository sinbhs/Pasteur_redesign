## About This Project
- 이곳은 **파스퇴르 기업 웹페이지 리뉴얼** 문서 저장소입니다.
- 해당 페이지는 디바이스에 따라 최적의 화면을 경험할 수 있도록 **반응형 페이지**로 제작되었습니다.
- 본 사이트는 상업적 용도가 아닌 **포트폴리오 목적**으로 리뉴얼 제작된 사이트이며 일부 이미지와 문구등의 저작권은 파스퇴르에 있음을 알립니다.
